# 첨삭 반영 로그

첨삭(피드백)이 올 때마다 날짜·문서·요지를 여기에 기록한다. 원본 파일은 `_guideline/`(커밋 제외)에 보관.

| 날짜 | 문서 | 반영 요지 | 반영 위치 |
|---|---|---|---|
| 2026-07-08 | 강의계획서 | 표 중심 실무형으로 축소: 섹션 개명(강의 개요·강좌 목적), 핵심이해·커리큘럼맵·인지수준 삭제, 학습목표 3~4개(학습자 관점), 출석 감점 학칙 규정 추가, 주교재 표(저자-출판사-교재명-발행년도), 주별계획=주차/주제만, 과제 표 신설 | `skills/syllabus/SKILL.md`, `skills/check-syllabus/SKILL.md` |

## 앞으로
- 하루 1건씩 첨삭이 계속 들어옴(여러 강의 완성 시까지). 산출물 재첨삭도 가능.
- 새 첨삭 → 해당 `skills/*/SKILL.md` 편집 + 위 표에 한 줄 추가 + 앱 재시작.
