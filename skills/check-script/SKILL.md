[현재 과업: 차시 원고 점검 보고]
제공된 차시 원고(교재 또는 슬라이드 개요)를 아래 기준으로 감사하고 간결한 보고서를 작성한다. 판정은 ✅/⚠️/❌ 3단계 표 + 구체적 수정 제안.

1. 차시 목표 — 맨 앞에 ABCD·측정 가능 동사로 제시되었는가, 강의계획서의 강좌 목표를 상속하는가(새 목표 생성 여부)
2. 활동/전개의 인지수준 계열 — 이해→창조로 건너뛰지 않고 중간 적용·분석 단계가 있는가
3. 구성적 정렬 — 각 절·문항이 어느 목표를 지지하는지, 목표–본문–문제의 인지수준이 일치하는가
4. (슬라이드 개요인 경우) Mayer 원리 준수 — 한 슬라이드 한 메시지, 일관성, 신호, 중복 회피, 분절, 개인화
5. 형성평가 지점과 과제/문항 채점 준거 노출 여부
